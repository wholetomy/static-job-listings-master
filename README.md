# Static Job Listings Master

Projeto desenvolvido como solução para um desafio do Frontend Mentor.

## Sobre

Aplicação que consome um `data.json` e permite:

- listar vagas de emprego
- adicionar e remover filtros para selecionar melhor as vagas.

---

## Tecnologias

- React
- TypeScript
- Vite
